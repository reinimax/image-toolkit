    This file defines available formats for returning a processed image. New functions added here will be automatically picked up when the function is added to the ALLOWED_FORMATS dictionary.

    Function arguments will be automatically passed when the function is called, if they were given via the API. The keys provided in the return_as object must match the parameter names of the Python function. For example, giving this JSON to the API

    "return_as": {
        "format": "webp",
        "quality": 20
    }

    would result in this function call: webp(image, buffer, quality=20).

    Note that each function automatically gets an image object of class PIL.Image.Image and a buffer object of class io.BytesIO passed as the first two parameters. The image should be saved to this buffer.

27def jpeg(image:Image.Image, buffer:BytesIO, quality=75) -> None:
28    """#Convert image to jpeg format.
29
30    **Parameters:**
31    - image (Image): PIL image object
32    - buffer (BytesIO): Buffer to which the image is written
33    - quality: Quality of the image. Shoulb be between 0 (lowest quality) and 95 (highest quality).
34    """
35    # If we get a string or float, convert to int. Pillow is picky and accepty only an int here.
36    quality = validate_int(quality, 75, 0, 95)
37    image.save(buffer, format="jpeg", quality=quality)

40def png(image:Image.Image, buffer:BytesIO, compress_level=6, optimize:bool=False) -> None:
41    """#Convert image to png format.
42
43    **Parameters:**
44    - image (Image): PIL image object
45    - buffer (BytesIO): Buffer to which the image is written
46    - compress_level: Must be between 0 and 9. 0 means no compression, 9 means maximum compression.
47    If `optimize`is true, `compress_level` is always 9.
48    - optimize (bool): If true, the image will be made as small as possible.
49    """
50    compress_level = validate_int(compress_level, 6, 0, 9)
51    if not isinstance(optimize, bool):
52        optimize = False
53    image.save(buffer, format="png", compress_level=compress_level, optimize=optimize)

56def webp(image:Image.Image, buffer:BytesIO, quality=80, lossless:bool=False) -> None:
57    """#Convert image to webp format.
58
59    **Parameters:**
60    - image (Image): PIL image object
61    - buffer (BytesIO): Buffer to which the image is written
62    - quality: Quality of the image. Shoulb be between 0 (lowest quality) and 100 (highest quality).
63    If `lossless`is true, this will instead determine the speed of saving the image (0 = fastest, 100 = slowest).
64    - lossless (bool): Whether to use lossless compression.
65    """
66    # If we get a string, try to convert to float
67    if isinstance(quality, str):
68        try:
69            quality = float(quality)
70        except:
71            # If an invalid argument was provided, use the default instead.
72            quality = 80
73    # Make sure quality is within the supported range
74    if quality < 0 or quality > 100:
75        quality = 80
76    if not isinstance(lossless, bool):
77        lossless = False
78    image.save(buffer, format="webp", quality=quality, lossless=lossless)
